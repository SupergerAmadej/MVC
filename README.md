# MVC
An MVC web application for a wine company. It supports the user adding different wines to different wine cellars and keep track of number of items. Authentication and authorization is based on .NET CORE IDENTITY, but it also supports 3rd party authentication (Google), database manipulation is managed with EntityFrameworkCore
